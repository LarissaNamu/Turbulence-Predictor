# Turbulence-Predictor


Formula for the Turbulence Index: \
T=(5/4)w+(3/7)i+(1/10)e \
T = Turbulence Index \
w = wind speed (mph)\
i = weather intensity \
e = |descending elevation difference (meters)| 

0 to 56 : green \
57 to 145 : yellow \
146+ : red 

green : none to slight tremors \
yellow : turbulence present \
red : severe turbulence
